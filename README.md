# 📚 College Event Booking System

A simple and interactive web-based application that allows users to explore college events and book seats easily. This project demonstrates front-end development using HTML, CSS, and JavaScript.

## 📌 Project Overview

The *College Event Booking System* is designed to help students:

- Browse upcoming events  
- View detailed event information  
- Book seats online  
- Login for access (basic validation)

## ⚙️ Features

### 🏠 Home Page
- Introduction to the system  
- Featured events  
- Navigation bar  
- Dark mode toggle  
- Current date display  

### 📅 Events Page
- List of all events  
- Search functionality  
- Event cards with details  

### 📄 Event Details Page
- Full event description  
- Date, time, and venue  
- “Book Now” button  

### 📝 Booking Page
- Booking form  
- Input validation  
- Confirmation message  

### 🔐 Login Page
- Username & password validation  
- Error/success messages  

### ℹ️ About Page
- Project information  
- Objectives & features  
- Animated counters
