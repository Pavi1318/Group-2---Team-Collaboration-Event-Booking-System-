About Our Project

About Project

The College Event Booking System is a simple web application designed to help students explore and book college events easily. It provides a simple and user-friendly platform to view event details and reserve seats.

Objectives

  • Make event booking simple for students
  • Provide clear event information
  • Reduce manual event registration

Features

  • Browse available events
  • View event details
  • Book seats online
  • Simple login system

Event Categories

  • Workshops (AI, Web Development)
  • Technical Events (Hackathons)
  • Cultural Events (Dance, Music)
  • Seminars & Career Talks

How It Works

  1. Login to your account
  2. Browse available events
  3. View event details
  4. Book your seat
